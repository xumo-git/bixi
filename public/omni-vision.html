灵瞳写轮眼

OmniVision · 万花筒

功能摘要

灵瞳写轮眼是墨码的「眼睛和手」——一套跨平台的桌面自动化引擎，集 macOS 原生操控、Android ADB 控制、Playwright 浏览器驱动、VLM 视觉闭环于一身。它让 AI 不再是盲人摸象，而是能看见屏幕、理解界面、精准操作的全能数字体。

macOS 原生操控

11 种桌面动作：click / drag / scroll / type / hotkey / menu / app_launch / app_kill / window_focus / screenshot / clipboard。支持 Accessibility API 和 CAEvent 双路径输入。

VLM 视觉闭环

截图→VLM分析→决策→执行→再截图。利用 Qwen3-VL / GPT-4o 等视觉模型，实现「看屏幕→理解→操作→验证」的完整闭环。

Android 移动操控

通过 ADB 连接 Android 设备，支持 tap / swipe / type / screenshot / app_launch。从电脑控制手机，全自动化测试与操作。

Playwright 浏览器

基于 Playwright 的浏览器自动化。页面导航 / 点击 / 表单填写 / 内容抓取 / 截图，Web 端全能操控。

实现方式

Python 模块化架构。`action_executor.py` 统一调度四算子（macOS / ADB / Playwright / VLM），`vlm_agent.py` 对接硅基流动等视觉模型 API，`adb_operator.py` 封装 Android 调试桥命令，`browser_operator.py` 提供 Playwright 高级接口。入口 `lingtong.sh` Shell 脚本提供终端 CLI。配套 `test_*.py` 验证各算子可靠性。

为什么做这个

灵瞳写轮眼的诞生源于一个朴素的痛点：AI 能写代码、能聊天、能推理，但「看不见屏幕」。当需要操作一个 GUI 应用、填写一个网页表单、或者从手机屏幕上抓取数据时，AI 像个盲人——只能靠猜。 万花筒 v2 版本将四个「写轮眼」合而为一：macOS 原生、Android ADB、Playwright 浏览器、VLM 视觉。每次操作都是看→想→做→检查的闭环。 最得意的场景：东方财富 App 的股票数据抓取。灵瞳通过 ADB 操控手机屏幕，截图传给 Qwen3-VL 分析，提取板块涨幅、持仓估值——整个过程全自动，比手动翻手机快 10 倍。
