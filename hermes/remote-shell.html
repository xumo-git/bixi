xumo-PC 远程连接

Remote Shell · 千里之外

功能摘要

SSH 远程连接 xumo-PC（Dell G5 5587 / i7-8750H / 31GB / GTX 1050Ti / Ubuntu 24.04）的完整技能。远程执行命令、跑 Docker/Dify、管理文件、利用 GPU 做轻量 AI 推理。双连接路径自动切换：WiFi（192.168.31.11）和直连网线（172.16.0.2）。

双连接路径

WiFi 连接优先，网络不通时自动切换到直连网线路径。两路 SSH 免密登录，无缝切换。

Docker 舰队管理

远程操控 Docker 容器：Dify / PostgreSQL / Redis / Weaviate / ssrf_proxy 全线管理。

GPU 推理

利用 GTX 1050Ti 做轻量 AI 推理。本地模型部署 / 推理加速 / 视频处理。

文件远程管理

SCP / rsync 文件传输，目录结构维护，日志远程查看。从 Mac 到 Linux 的文件通道。

实现方式

纯 Agent 技能（SKILL.md）。SSH 密钥免密登录。双路径连接 + Docker 命令封装 + GPU 环境配置指南。

为什么做这个

xumo-PC 是一台 Dell G5 5587 游戏本，装着 Ubuntu 24.04。它是墨家军的「算力外挂」——跑着 Dify、PostgreSQL、Weaviate 向量数据库等基础设施。 但问题来了：它不在本地，需要远程连接。WiFi 可能断，内网可能变，IP 可能换。 双连接路径是最得意的设计：WiFi 连接 192.168.31.11，直连网线走 172.16.0.2。一个不通自动换另一个，用户无感知。 这不仅仅是个 SSH 技能——它是整个墨家军基础设施的「远程手」。没有它，Dify 知识中台就搭不起来，Agent 的知识检索能力就大打折扣。
