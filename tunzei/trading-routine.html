工作日技能

Trading Routine · 盯盘三件套

功能摘要

A 股工作日例行三件套自动化：盘前简报（9:20 集合竞价分析）、盘中方向（11:00 板块轮动研判）、收市总结（15:30 持仓估值更新）。通过灵瞳操控东方财富 App 获取实时行情，存钱罐思维驱动，不追涨杀跌。335 行完整技能定义。

集合竞价分析

9:20 自动分析集合竞价数据。高开/低开/平开判断，隔夜情绪评估，今日策略预判。

板块轮动追踪

11:00 扫描板块涨幅榜。主力资金流向、板块轮动节奏、热点持续性评估。

持仓估算

15:30 更新全部持仓估值。当日盈亏/累计盈亏/仓位占比/ETF 估值百分位。

灵瞳操控

通过灵瞳写轮眼操控东方财富 App 获取行情。截图→VLM 分析→数据提取，全自动数据采集。

实现方式

纯 Agent 技能（335 行 SKILL.md）。`task=premarket/intraday/closing` 参数化。通过灵瞳操控东方财富 App。含重试逻辑、模型降级、6 个 test case JSON。

为什么做这个

炒股最怕什么？凭感觉。 工作日技能把「凭感觉炒股」变成了「按流程操作」。每天三个固定时点，做三件固定的事，输出三份结构化的报告。 没有情绪化决策的空间——规则写在技能里，Agent 照着执行。开盘就是分析、盘中就是跟踪、收盘就是总结。 最酷的是通过灵瞳操控东方财富 App：Agent 能在手机上打开东方财富、截图、分析、提取数据。这比任何 API 都灵活——因为灵瞳能操作的是完整的 App，而不是有限的数据接口。
