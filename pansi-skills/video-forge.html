视频合成

Video Forge · 文生视频

功能摘要

小说剧本→竖屏短视频的全自动流水线。文生图（本地 SDXL API）+ 情感 TTS（ChatTTS）+ FFmpeg 合成三合一。支持多种情绪语音和古风/现代/仙侠提示词模板。小说段落直接变成短视频。

文生图

本地 SDXL API（:8189）根据剧本内容自动生成配图。古风/现代/仙侠多风格提示词模板，画面与内容风格一致。

情感 TTS

ChatTTS（:19527）生成情感语音朗读。喜怒哀乐多种情绪语音可选，小说对白的角色感拉满。

FFmpeg 合成

图片 + 语音 + 字幕 → 竖屏短视频。FFmpeg 自动合成，输出抖音/视频号格式。

OpenAI 兼容 API

文生图服务兼容 OpenAI API 格式。任意 OpenAI 客户端可直接调用本地 SDXL 能力。

实现方式

Python 服务模块：`servers/image_server.py`（文生图，OpenAI 兼容 API，端口 8189），`tts_server.py`（ChatTTS 情感语音，端口 19527），`compose_video.py` + `make_video.py`（FFmpeg 合成）。

为什么做这个

小说写好了，怎么让它「动起来」？ 视频合成技能是盘丝洞的「影视化」尝试：把每一章小说段落变成一条短视频。文生图做画面，ChatTTS 做语音朗读，FFmpeg 做最终合成。 全本地运行——SDXL 跑在本地 GPU 上，ChatTTS 也是本地服务。不需要第三方 API，没有按次付费的压力。 最让我激动的是「情感 TTS」：ChatTTS 能生成带情绪的语音，高兴时语调上扬，悲伤时语速放慢。小说里的对白有了真实的「语气」，读者体验完全不同。
