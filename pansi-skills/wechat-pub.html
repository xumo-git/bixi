盘丝技能

公众号发布

WeChat Pub · 粉丝直达

功能摘要

微信公众号自动发布系统。独立于 GitHub Pages 部署，将小说章节直接发布到公众号草稿箱。支持袁天罡 / 天命五九两书发布（蕊的故事禁止发布）。wenyan 渲染排版，多主题可配置。

公众号 API 集成

通过微信公众平台 API 操作草稿箱。AppID + AppSecret 鉴权，IP 白名单安全防护。

wenyan 排版

文言/水墨风格排版引擎。多主题可选（水墨 / 古风 / 极简），公众号阅读体验拉满。

权限控制

明确的书目发布规则：袁天罡和天命五九可以发公众号，驯蕊禁止发布——保护角色隐私。

实现方式

Shell 脚本 `publish_to_wechat.sh` + 纯技能。通过微信公众号 API 接口操作。wenyan 渲染引擎排版后推送到草稿箱。

为什么做这个

GitHub Pages 是给懂技术的人看的。读者们都在微信公众号里。 公众号发布技能解决的就是「最后一公里」：小说写好了，怎么送到读者手机里？ 直接操作微信公众号 API 的草稿箱功能——不是发布（发布需要人工审核），而是存为草稿。作者登录公众号后台看一眼，没问题的话手动发布。 wenyan 排版是点睛之笔：用文言风格的 CSS 渲染 HTML，匹配小说的古风气质。读者打开文章的第一眼，就知道这是一篇有格调的小说。
