盘丝技能

总结计划

Weekly Planner · 深度规划

功能摘要

每周六执行的深度规划器。三本书通用——总结本周写作进展 + 诊断现状 + 规划下周每天每时段的详细写作指令。228 行完整技能定义，让写作有条不紊。

本周总结

自动收集本周写作数据：章节数 / 字数 / 完成率。对比周计划评估达成度。

现状诊断

分析写作节奏：哪些章节写得好、哪些卡住了、角色塑造哪里需要调整。深度诊断替代盲目推进。

下周规划

每天每时段的详细写作指令。上午写什么、下午写什么、晚上写什么——精确到章节和字数。

三书通用

同一套规划模板适配三本书。配置不同的 RULES.md / 写作进度 / 时间线 / 大事年表驱动。

实现方式

纯 Agent 技能（228 行 SKILL.md）。分阶段执行：收集本周数据 → 诊断问题 → 规划下周 → 输出周报 + 周计划。操作 RULES.md / 写作进度 / 时间线 / 大事年表等文件。

为什么做这个

写作最怕的不是写不出来，而是不知道写什么。 总结计划技能就是解决「不知道写什么」这件事的。每周六，它自动做三件事： 1. 这周写了什么？进展如何？ 2. 哪里写得顺？哪里卡住了？怎么改进？ 3. 下周怎么写？每天每时段写什么？ 没有这个规划器，日常写作就是盲人摸象——今天想到什么写什么，情节前后不搭，节奏忽快忽慢。 有了它，三本书的写作变成了一条有序的流水线：周六规划→周一到周五按计划执行→周六复盘总结→再规划下周。闭环运转，持续产出。
