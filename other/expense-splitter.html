报销分赃

Expense Splitter · 分钱大师

功能摘要

差旅费用审核与分账统计工具。读取标准化命名后的发票 PDF 文件名，自动进行行程合规性检查、费用分类统计（火车票 / 的士 / 补助）、按人员拆分报销明细。输出三件套：异常报告 + 费用汇总 + 个人明细。出差补助 80 元/天自动计算。

合规检查

自动检查行程是否完整（去程→回程）、票据是否齐全、日期是否合理。输出异常报告。

费用分类

自动区分火车票 / 的士 / 打车 / 住宿 / 补助。分类汇总，一目了然。

按人分账

多人出差自动拆分。A 的发票、B 的发票、C 的补助——各算各的，清清楚楚。

补助自动计算

出差补助 80 元/天。根据行程日期自动计算天数，自动生成补助金额。

实现方式

Python 脚本 `报销汇总脚本.py` + PyInstaller 打包的二进制 `报销分赃工具`。只读文件名（依赖发票重命名技能预处理），不读 PDF 内容。出差补助 80 元/天公式硬编码。

为什么做这个

发票重命名做好后，下一个问题来了：怎么分账？ 几个人一起出差，A 买了火车票、B 打了的士、C 付了酒店——最后要算每个人花了多少钱、公司要报销多少、每个人还要给对方多少钱。 报销分赃技能就是做这个的。它只读文件名——因为发票重命名技能已经把所有信息写进文件名了。金额、日期、人名、路段——全在文件名里。 行程合规检查是最有价值的功能：有没有缺了回程票？补助天数算对了没？日期有没有矛盾？——所有可能的错误一次性检查完毕。 输出三件套：异常报告（有问题的先处理）、费用汇总（总共花了多少）、个人明细（各人各算各的）。一份 Excel 表格发群里，大家确认没问题，直接提交财务。
