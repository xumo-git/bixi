发票重命名

Invoice Linter · 票据管家

功能摘要

发票/行程单 PDF 智能重命名工具。自动读取 PDF 内容并按键生成标准文件名。支持三大类票据：高德打车行程单 / 发票、铁路电子客票、出差申请表。命名格式：`26_{金额}_{类型}_{路段}_{人名}.pdf`。

三类票据识别

自动识别票据类型：高德打车电子发票/行程单、12306 铁路电子客票、出差申请表。不同类型不同解析策略。

地址映射引擎

内置 ADDR_MAP 地址名称映射表。「广州白云站→广州」、「站→去掉」等规则。标准化路段名。

PDF 内容解析

直接读取 PDF 文字内容提取关键字段：金额、日期、起止地点、人名。无需 OCR，原生 PDF 解析。

PyInstaller 打包

打包为独立二进制程序，无需 Python 环境即可运行。Windows/macOS 双平台分发。

实现方式

Python 脚本 `validate_invoice_name.py` + PyInstaller 打包的二进制 `发票重命名工具`。PDF 内容解析 + 地址映射表 + 格式组装。

为什么做这个

出差报销最烦的是什么？整理发票。 一堆 PDF 文件，名字全是乱码或扫描件编号。要一个个打开看是什么票据、多少钱、哪天发生的、谁的名字。然后手动重命名。 发票重命名技能就是解决这个痛点的：双击运行→自动扫描文件夹内所有 PDF→读取内容→智能重命名。10 秒搞定 50 张发票。 地址映射引擎是最用心的设计：「广州白云站」自动缩写成「广州」，「深圳北站」变成「深圳北」。规则写死在映射表里，遇到新的地址格式随时扩展。 打包成独立二进制是最务实的决定：不是每个人都有 Python 环境。双击就能运行，才是真正的工具。
